Master of Magic clone

* Source language: golang, https://go.dev/
* Graphics library: ebiten, https://ebitengine.org/

Run/Build:

```
$ go get -u ./game/magic
$ go run ./game/magic
```

```
$ go build -o magic ./game/magic
```
or
```
$ make
```
